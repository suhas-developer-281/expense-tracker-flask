# Expense Tracker Web Application

A beginner-friendly Expense Tracker web application built using Python Flask, HTML, and CSS.

This project focuses on building a clean project structure, connecting frontend pages with a Flask backend, and serving static files correctly.

---

## Features (Current Version)
- Flask backend setup
- HTML templates rendered using Flask
- CSS styling using Flask static folder
- Clean and responsive expense input form UI

---

## Technologies Used
- Python
- Flask
- HTML
- CSS

---

## Project Status
This project is currently in progress.

Future enhancements planned:
- Handling form submission
- Database integration (SQLite)
- Displaying expense list
- Total expense calculation

---

## How to Run the Project

1. Install Flask:
pip install flask

2. Run the application:
python app.py

3. Open browser and visit:
http://127.0.0.1:5000/
